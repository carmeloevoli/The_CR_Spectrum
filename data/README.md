## Data

Most charged-particle data used by the plot are read from the linked `kiss_tables/` directory. These tables are primarily based on data collected from CRDB and KCDC.

Credit:

- CRDB: https://lpsc.in2p3.fr/crdb
- KCDC: https://kcdc.iap.kit.edu

The files in `data/source/` contain original gamma-ray and neutrino tables. The scripts in this directory convert them into plot-ready files in `data/output/`. Detailed references for the source tables will be added here later.
